## Pokedex
- This repository store code about Pokedex application using in representation of Java class @UIT
- Using gradle to manage third party dependency

```shell
$ git clone https://github.com/sonlhcsuit/pokedex-jfx.git
$ cd pokedex-jfx
$ gradle build
$ gradle run
```
